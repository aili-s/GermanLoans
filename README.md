# GermanLoans
Review and explore of loan data in Germany. URL: https://archive.ics.uci.edu/dataset/144/statlog+german+credit+data 

This dataset is categorical and encoded. When writing SQL queries, the focus was on extracting the necessary information from the data, while when performing the task in Excel, the main task was to transform the encoded data into readable and presentable data
